to compile and run the program...

- go on cmd/terminal
- cd to this folder
- compile and run based on your compiler

examples:

visual studio:
>	cl problem1.cpp
>	problem1.exe
	
mac/unix:
>	make problem1
>	./problem1

linux:
>	g++ problem1.cpp
>	./a.out



[PROBLEM 1]
Once the program is running, type a paragraph up to 100 words. It will then return statistics:
-the least frequent letter
-all unique words and their frequencies
-the most frequent word

see p1example



[PROBLEM 2]
Once the program is running, enter numbers separated by spaces. It will then return back the numbers sorted from
least to greatest. It will also show what you originally inputted.

see p2example
